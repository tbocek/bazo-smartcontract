\begin{abstract}
	\thispagestyle{main}
	Das Ziel der Bachelor Arbeit war die Integration von Smart Contracts in die Bazo Blockchain. Das Ergebnis ist eine Blockchain auf der nach einiger Einarbeitungszeit einfachere Smart Contracts ausgeführt werden können. 
	
	
Smart Contracts sind Instruktionen die in Transaktionen einer Blockchain geichert werden.  Eine Blockchain kann in diesem sinne als dezentrales Buchführungssystem verstanden werden, von dem multiple gleichartige und unveränderbare Kopien existieren. Diese Arbeit befasst sich mit der Implementation und Integration eines Interpreters bzw. einer Virtuellen Maschine für Smart Contracts. Als resultat können Transaktionen erstellt werden die einen Smart Contract enthalten und Transaktionen, die die Ausführung des Codes eines Smart Contracts bewirken, welcher dann von der Virtuellen Maschiene ausgeführt wird und ein entsprechendes Resultat im State des Miners speichert.
