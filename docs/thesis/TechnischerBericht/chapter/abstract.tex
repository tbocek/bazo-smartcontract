%Welches Problem wird mit der Untersuchung gelöst?
This thesis is concerned with the integration of smart contracts into the Bazo Blockchain with the goal of creating a platform for  decentralized applications. The Bazo Blockchain started as a research project at the University of Zurich in cooperation with a financial service provider and is extended independently by Prof. Dr. Thomas Bocek. 
%Warum ist die Untersuchung wichtig?
Smart contracts are programs that are stored and called by transactions on a blockchain. Smart contracts offer opportunities in automation by  triggering transactions on certain events and bring along advantages provided by blockchains such as immutability, public visibility of transactions and decentralization. As blockchains are trust-less protocols, it is guaranteed that smart contracts are executed as intended. 
%Wie wird das Problem gelöst?
The integration of smart contracts was solved by implementing a virtual machine that executes the instructions carried by a transaction. Furthermore the virtual machine was embedded within the mining application, which required to alter the blockchain protocol and mining application itself. 
%Welches sind die wichtigsten Ergebnisse? 
As a result, smart contracts can be deployed and transactions that call functions of smart contracts can be executed. Calling a smart contract function leads to the execution of the contract in the virtual machine and persisting the result as updated state variable. 
%Wie sind sie zu bewerten?
The Bazo Blockchain continuous to be a research project, this means that it requires a lot of training to use the corresponding functionalities. 
%Was bedeutet das für die künftige Arbeit in diesem Feld?	
Follow-up theses could simplify the development of smart contracts for the Bazo Blockchain by creating a high-level programming language that can be compiled to Bazo virtual machine instructions.
